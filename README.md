***DSA APPLICATIONS AND WEATHER APP***

A curated collection of real-world Data Structures & Algorithms projects built using Python, C++, and Web technologies, focusing on visualization, problem-solving, and practical understanding rather than just theory.

This repository demonstrates how core DSA concepts are applied in interactive applications, games, and visual tools.

🚀 Projects Included
🧩 1. Sudoku Game & Solver

📁 Folder: sudoku_game

A complete Sudoku game with an automated solver and visualization.

🔍 Features

Interactive Sudoku gameplay

DFS + Backtracking based solver

Step-by-step solving visualization

Clean modular Python design

🧠 Concepts Used

Recursion

Depth-First Search (DFS)

Backtracking

2D arrays

Constraint validation

📌 Great for understanding recursive problem solving.

🧠 2. Algorithm Visualizer – Maze Runner

📁 Folder: algorithms_maze

An interactive maze-solving visualizer that demonstrates how different pathfinding algorithms explore a graph.

🧠 Algorithms Implemented

Breadth-First Search (BFS)

Depth-First Search (DFS)

A* (A-Star)

✨ Features

Real-time algorithm visualization

Step-by-step node traversal

Clear comparison of algorithm behavior

Interactive UI

🧠 Concepts Used

Graph traversal

Heuristics (A*)

Queues & stacks

Time–space tradeoffs

📌 Ideal for visual learners studying graph algorithms.

🌦️ 3. Weather App (Web Project)

📁 Folder: Weather_App

A modern JavaScript-based weather application with animated UI and real-time data.

✨ Features

Live weather data using OpenWeatherMap API

Weather-based animations (rainy, cloudy, sunny, cold)

Geolocation support

Clean UI/UX with background transitions

🧠 Concepts Used

API integration

Asynchronous JavaScript

DOM manipulation

Configuration-based design

Frontend security practices

⚠️ Note: Requires running via a local server (http://localhost) for geolocation to work.

🧠 Focus Areas
